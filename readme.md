# 🛡️ NTZ PROTO

Welcome to NTZ PROTO! We are building a super-secure VPN tunnel to keep your internet traffic safe from future supercomputers. 🚀
📖 What is this project?

NTZ PROTO is our experimental VPN project. We want to protect data using brand-new security methods known as post-quantum cryptography. Basically, we are creating a custom network tunnel that even the most advanced computers will not be able to crack!
✨ What is new in 1.0v

    🔌 We successfully built a basic TCP server and client to talk to each other.

    🧱 This setup gives us a solid starting point to add heavy-duty encryption later.

    🧩 The code is kept simple so we can easily test and add new pieces as we go.

🗺️ Our plan for the future

    🛠️ Step one is making sure our basic TCP connection is rock solid.

    🔐 Step two is plugging in our super-strong encryption.

    🌐 Step three is getting the tunnel to work with TUN interfaces and UDP traffic.

    ⚡ Step four is making everything run really fast on any computer.

🚀 How to try it out

We are still putting the finishing touches on our setup process. Check back soon for easy instructions on how to download, build, and run the code yourself!
